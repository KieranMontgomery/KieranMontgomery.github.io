Detailed Description

Function for saving data variable(s) in VTK-legacy standard. VTK_VAR is an interface to 8 different functions, there are 3 functions for scalar variables, 3 functions for vectorial variables and 2 functions texture variables: scalar and vectorial data can be R8P, R4P and I4P data while texture variables can be only R8P or R4P. This function saves the data variables related to geometric mesh.

Remarks
The inputs that must be passed change depending on the data variables type.
Note
Examples of usage are:
Scalar data calling:
...
integer(I4P):: NN
real(R4P):: var(1:NN)
...
e_io=vtk_var(nn,'Sca',var)
...
Vectorial data calling:
...
integer(I4P):: NN
real(R4P):: varX(1:NN),varY(1:NN),varZ(1:NN)
...
e_io=vtk_var('vect',nn,'Vec',varx,vary,varz)
...
